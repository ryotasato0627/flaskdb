# FlaskDB

## 🧭 プロジェクト概要
Flaskを用いてWebアプリとAPI開発の違いを理解することを目的とした学習プロジェクトです。  
バックエンドの基本構造（MVC / レイヤードアーキテクチャ）を実装しながら、  
セッション管理やトークン認証などのWebアプリ基礎を習得することを目指しています。

---

## 🎯 目的
- Pythonによるバックエンド開発の基礎を習得  
- WebアプリとAPIの違いを明確に理解する  
- サーバーサイドのレイヤー構造（Routes / Models / Services）を理解する  
- FlaskからFastAPIへの移行を通じてフレームワークの本質を学ぶ
